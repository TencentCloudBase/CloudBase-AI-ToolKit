## CodeBuddy IDE 版

**CodeBuddy IDE** 用户请直接在设置页/集成点击"Tencent CloudBase"管理按钮并授权，无需手动配置 MCP。

## CodeBuddy 插件版

### 通过 MCP 市场安装（推荐）

1. 点击 CodeBuddy 右上角的 MCP 按钮
2. 在 MCP 市场中搜索 CloudBase
3. 点击安装即可

## 常见问题

**Q: MCP 连接失败？**
A: 检查配置文件格式是否正确，重启 CodeBuddy，确认网络连接正常。

**Q: 工具数量显示为 0？**
