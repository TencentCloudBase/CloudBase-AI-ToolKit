## 🚀 通用配置流程

所有 IDE 的配置流程都类似，只需 4 步：

```mermaid
graph LR
    A[📦 获取项目模板] --> B[🔧 配置 MCP]
    B --> C[📋 启用 AI 规则]
    C --> D[🎯 开始开发]

    style A fill:#e1f5fe
    style B fill:#e8f5e8
    style C fill:#fff3e0
    style D fill:#f3e5f5
```

1. **📦 获取项目模板** - 下载预配置的项目模板
2. **🔧 配置 MCP** - 添加 CloudBase MCP 服务器
3. **📋 启用 AI 规则** - 使用云开发最佳实践规则
4. **🎯 开始开发** - 对 AI 说"登录云开发"开始使用
