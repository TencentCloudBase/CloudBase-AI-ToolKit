# MCP

使用 Model Context Protocol (MCP) 将 CloudBase MCP 连接到你的 AI 开发工具。连接后，AI 助手可以代表你与 CloudBase 项目交互和查询。

## 安装

### 步骤 1：配置你的 AI 工具

选择你的 AI 开发工具，然后按照安装说明操作：

<IDESelector />

### 步骤 2：验证连接

配置完成后，在 AI 对话中输入以下内容验证：

```
检查 CloudBase 工具是否可用
```

## 手动确认工具调用

大多数 MCP 客户端（如 Cursor）会在执行每个工具调用前要求你手动确认。我们建议你始终启用此设置，并在执行前仔细查看工具调用的详细信息。

## 下一步

你的 AI 工具现在已连接到 CloudBase 项目。尝试让 AI 工具使用自然语言命令查询你的数据库。


## 前置条件

### Node.js 环境

确保已安装 Node.js v18.15.0 或更高版本：

```bash
node --version
```

如果未安装，请从 [Node.js 官网](https://nodejs.org/) 下载安装。

### 云开发环境

访问 [腾讯云开发控制台](https://tcb.cloud.tencent.com/dev) 开通环境。新用户可以免费开通体验。

## 开始使用

配置完成后，在 AI 对话中输入：

```
登录云开发
```

AI 会自动引导你完成登录和环境选择。然后你可以开始使用自然语言描述需求，让 AI 帮你开发应用。

## 示例

```
帮我创建一个待办事项应用，包含：
- 添加、删除、编辑待办事项
- 数据存储到云数据库
- 支持标记完成状态
- 部署到云端
```
